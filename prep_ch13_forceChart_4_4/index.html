The data set above could be used for force diagram because it contains the network data that consists of nodes and edges. This data set is called a graph. The force layout allows you to simulate movements based on physics. As the book talks about, the dataset can contain other attributes than nodes and edges. Therefore, the following two dataset can also be used for force diagram. Using these datasets, we can create things such as draggable nodes, graph, and many other visualization using physics.
